# AjaxFormSample

RailsのFormヘルパであるform_withを使ったajax非同期通信を簡単なタスクアプリを作りながら実装。  
タスクの追加と削除が非同期でできます。  
あくまでform_withの特徴を把握したいがための開発なので、タスクアプリとしての厳密性は重視していない仕様です。

#### 言語バージョン
* Ruby 2.6.4

#### フレームワーク
* Ruby on Rails 6

#### ライブラリ
* jQuery 3.5.0

#### DB
* MySQL

#### 使用した技術
* haml-rails (gem)
* webpack
